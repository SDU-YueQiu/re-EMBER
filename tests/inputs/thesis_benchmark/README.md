# Thesis benchmark inputs

These OBJ files are converted from the thesis experiment OFF caches under `Shandong-University-Undergraduate-Thesis-Design-Template-main/experiments/results/benchmark/cached_inputs/off`.

The set contains five medium and five large boolean input pairs. Each pair keeps the original experiment pair id and contains `lhs.obj` and `rhs.obj`. See `manifest.csv` for source paths and face counts.
